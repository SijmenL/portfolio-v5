Park Plaag

Park Plaag is een Tower Defense game die zich afspeelt in het Rotterdamse Park 1943, waar je met stadsiconen ongedierte tegenhoud.Park Plaag (Park Plague) is a Tower Defense game that is set in the Rotterdam park 1943, in which you have to use the city's hero's as pest control. (🇳🇱)

Begrijpbaar

Begrijpbaar is een systeem waarbij mensen met auditieve beperking toch alle informatie tijdens een dagje uit meekrijgen.Begrijpbaar (Understandable) is an interactive system which helps people with a hearing impairment understand information during a day out (🇳🇱)
